# SoulShelf Flutter App 📱

> Personal Media Tracker & Digital Journal Mobile Application

SoulShelf is a Flutter-based mobile application that helps users track books, songs, movies, TV shows, and anime while also managing personal journals, planners, and reflections in one secure platform.

The application is connected to a Laravel REST API backend with offline support and secure authentication.

---

# ✨ Features

* 📚 Track Books, Songs, Movies, TV Shows & Anime
* 📝 Personal Journal & Daily Planner
* 🔒 PIN Protected Private Space
* 🎨 Light & Dark Theme
* 🤖 AI Chatbot Prototype
* ⭐ Rating & Recommendation System
* 📁 Collections Management
* 🔄 Offline Support & Auto Sync
* 📸 Cover Image Uploads
